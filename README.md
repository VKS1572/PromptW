PromptWarsProject
Description

This repository contains my PromptWars submission using Google Antigravity. It demonstrates how prompts can be processed using a Python script, with outputs saved for review.

The project includes:

Example prompt file
Python script to process the prompt
Result file containing processed output
Repository Structure
PromptWarsProject
├─ scripts
│   └─ process_prompt.py       # Python script to process prompt
├─ prompts
│   └─ prompt_1.txt            # Input prompt file
└─ results
    └─ output_1.txt            # Processed output
Prerequisites

Before running the script, ensure:

Python 3.x is installed
prompt_1.txt exists in the prompts folder
How to Run
Open a terminal and navigate to the PromptWarsProject folder:
cd C:\Users\vikas\PromptW\PromptWarsProject
Run the script:
python scripts/process_prompt.py
The output will be saved in the results folder as output_1.txt.
Notes
The script automatically creates the results folder if it doesn’t exist.
Make sure all work is done in a single branch for submission.
Repository size must remain under 1 MB.
